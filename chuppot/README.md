## 📸 הוספת מדיה

### תמונות
1. שמור את התמונות שלך בתיקיה `assets/images/`
2. שם הקבצים: `ceremony-01.jpg`, `ceremony-02.jpg`, וכו'
3. גודל מומלץ: 1920x1280 פיקסלים
4. פורמט: JPG או PNG
5. גודל קובץ: מתחת ל-500KB (מטובות לאינטרנט)

**כדי להשתמש בתמונות:**
- פתח את `index.html`
- מצא את הסקציה `<!-- ===== GALLERY SECTION ===== -->`
- החלף את ה-placeholders בקוד הבא:

```html
<div class="gallery-item card fade-in">
    <img src="assets/images/ceremony-01.jpg" alt="תיאור התמונה" loading="lazy">
</div>
```

### סרטונים
1. שמור את הסרטונים שלך בתיקיה `assets/videos/`
2. שם הקבצים: `clip-01.mp4`, `clip-02.mp4`, וכו'
3. פורמט מומלץ: MP4 (H.264)
4. גודל: מתחת ל-50MB

**כדי להשתמש בסרטונים:**

```html
<div class="gallery-item card fade-in">
    <video controls poster="assets/images/video-thumbnail.jpg">
        <source src="assets/videos/clip-01.mp4" type="video/mp4">
    </video>
</div>
```

## 🚀 הפעלת האתר

### אופציה 1: פתיחה ישירה
פשוט לחץ כפול על הקובץ `index.html` והוא ייפתח בדפדפן.

### אופציה 2: שרת מקומי (מומלץ)
אם יש לך Node.js מותקן:

```powershell
# בתיקיית הפרויקט, הרץ:
npx -y serve .

# או:
npx -y http-server .
```

לאחר מכן פתח את הדפדפן בכתובת: `http://localhost:3000`
